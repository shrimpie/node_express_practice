Learn node.js and express.js, referencing to the tutorial of Ray Villalobos @Lynda.com.


Notes:

1. Why Node.js?
  * Node.js: runtime chrome
  * Server javascript
  * Asynchronous events
  * Non-blocking IO

2. Why express.js?
  * Streamlined Node.js
  * Adds server methods
  * Routing mechanism
  * Easier APIs
  * Middleware friendly
